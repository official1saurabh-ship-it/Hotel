# Hotel Management Software — Biosoftech

India's smart hotel management platform. Manage bookings, check-ins, housekeeping, POS billing, inventory, CRM, and GST — all from a single platform.

Built for Indian hotels, resorts, guest houses, and stays.

## Modules

- Front Desk & Room Booking
- Housekeeping & Maintenance
- POS Billing & Service Management
- Room & Rate Management
- Inventory & Asset Control
- Guest CRM & Loyalty
- Banquet & Events
- Financial Reports & GST (GSTR-1 / GSTR-3B)
- Staff Attendance
- Multi-Branch Support

## Tech Stack

- **Framework:** React 19
- **Routing:** React Router 7
- **Styling:** Tailwind CSS 4
- **Build:** Vite 7
- **Icons:** React Icons (Lucide)
- **Email:** EmailJS

## Getting Started

```bash
npm install
npm run dev
```

The dev server will start at `http://localhost:5173`.

## Build

```bash
npm run build
```

Output goes to the `dist/` folder.

## Preview

```bash
npm run preview
```
